```
npm install
```

```
node src/index.js
```

BlockRandom Telegram Bot converts blockchain hash into a random number 
```

h(S) = S[0] + S[1] * P + S[2] * P^2 + S[3] * P^3 + ... + S[N-1] * P^N-1

```
